SAE 3.02

Develop communicating applications: design of a distributed infrastructure with onion routing and end-to-end encrypted messaging in Python.

SAE Summary

During this SAE, we had to create an encrypted messaging system in Python, as well as create an "onion" routing infrastructure with a master server, routers, and clients. The goal was for the message to be encapsulated layer by layer using a "homemade" asymmetric encryption algorithm of the RSA type. We also produced extensive documentation to support this project.

View the project and its documentation on GitHub

Python (Sockets & Threads)

Development of the distributed architecture using object-oriented programming, management of network connections (Sockets), and parallelism (Threading).

PyQt6

Library used to develop the graphical user interfaces (GUI) for the various clients and the master's monitoring dashboard.

MariaDB

Maths & SymPy

"Onion" Encapsulation Logic

Here is an excerpt of our asymmetric encryption implementation (based on RSA). Since in this type of algorithm, the data size must not exceed that of the key (N), long multi-layer encapsulated messages posed a problem. I therefore implemented a chunk_message function to split the string into fixed-size blocks (based on the bytes of N) before encrypting them individually and concatenating them with separators. 

def chunk_message(message, block_size):
    """Split the message into blocks of the maximum key size (N)"""
    data = message.encode("utf-8")
    return [data[i:i+block_size] for i in range(0, len(data), block_size)]

def chiffrement(message, pubkey):
    """Encrypts a long message by cutting it first"""
    SEP_BLOCK = '$$$' 
    SEP_PART = '$#$'

    _, n = pubkey
    block_size = (n.bit_length() - 1) // 8 
    blocks = chunk_message(message, block_size)
    
    cipher_blocks, lengths = [], []
    for block in blocks:
        c, l = encrypt_block(block, pubkey) # pow(m, e, n)
        cipher_blocks.append(c)
        lengths.append(l)
        
    # Reconstitution of the final encrypted payload with preserved lengths
    message_chiffre_parts = []
    for i in range(len(cipher_blocks)):
        message_chiffre_parts.append(f"{cipher_blocks[i]}{SEP_BLOCK}{lengths[i]}")
        
    return SEP_PART.join(message_chiffre_parts)
