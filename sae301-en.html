SAE 3.01

Implement a transmission system: design of a smart connected plug and development of remote control interfaces.

SAE Summary

During this SAE, we had to create a model of a connected electrical plug using an ESP8266 board. This allows us to control our plug (an LED) via MQTT from a mobile application and a dynamic Django website.

Environment and Tools

ESP8266

Wi-Fi microcontroller used to control the plug's relay and communicate with the MQTT broker.

Python / Django

Framework used for creating the dynamic web server hosted on a Raspberry Pi.

MQTT

Lightweight messaging protocol (Pub/Sub) used for real-time communication between IoT and the web.

Android Application

Development and configuration of a mobile application for remote control of the plug.

My part: creating the dynamic website

During the project, I participated in the creation of the dynamic Django website. This was meant to implement the various functionalities of the plug:

Features implemented on the site

  • Turn the plug on and off manually.
  • Real-time retrieval of the plug's status (status feedback).
  • Scheduling daily on/off time slots.
  • Graphical display of temperatures received from the DS18B20 sensor.
  • Alert for exceeding the configured maximum temperature.

Main technical steps

1. Hardware Design

Circuit modeling via EasyEDA and integration of physical components (ESP8266 board, Dallas DS18B20 temperature sensor, relay) meeting cost and size criteria.

2. Embedded Development (C++)

Programming the microcontroller via the Arduino IDE. Wi-Fi connection management (AP then STA via WiFiManager) and publish/subscribe to MQTT topics (PubSubClient).

3. Back-End Infrastructure

Deployment of a private MQTT broker and creation of an MVC architecture using Django to interact with the MySQL relational database storing temperature readings and time slots.

4. Server Logic and Telemetry

Creation of Python automation scripts using the Paho-MQTT library to subscribe to plug requests, manage states (LedState), and trigger alert emails.

Validated Skills
